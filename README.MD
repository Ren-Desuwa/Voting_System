ELECTION SYSTEM - SETUP GUIDE
=============================

PREREQUISITES:
1. You must install "Docker Desktop" for Windows.
   Download here: https://www.docker.com/products/docker-desktop/
   (Install it and restart your computer).

HOW TO START:
1. Open Docker Desktop and let it finish loading.
2. Double-click "start_server.bat".
3. Wait about 15 seconds. The Admin Panel will open in your browser automatically.

HOW TO STOP:
1. Double-click "stop_server.bat" when the election is over.

ADMIN INFO:
- Admin URL: http://localhost:8080/admin.html
- Default PIN: 1234 (You can change this in the admin panel)

PUBLIC VOTING URL:
- The public link will appear in your Admin Panel under "Global Configuration".
- Share that link with the voting lab computers.
